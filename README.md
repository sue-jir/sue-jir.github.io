# 武汉跨境运营求职雷达

这是一个面向武汉跨境电商运营求职的本地/静态网页小工具。

用途：

- 按你的简历能力筛选武汉跨境电商运营岗位和公司。
- 数据拆成 `activeJobs`、`watchCompanies`、`searchEntrances` 三类。
- 公司卡片显示当前状态、最后核验时间、推荐动作和风险标签。
- “在招岗位”直接打开企业岗位页或岗位详情页；百度只保留在搜索入口区。
- 记录每家公司投递状态和备注，数据保存在浏览器本地。
- 粘贴招聘 JD，输出“建议投 / 先问 / 不建议投”、风险词和可问 HR 的问题。
- 复制导出当前名单，方便发给自己或继续整理。

打开方式：

直接用浏览器打开 `index.html`。不需要安装依赖。

数据说明：

公司池来自 2026-05-27 可直达的公开招聘页、校园就业网、企业介绍页、武汉跨境电商产业公开信息和招聘平台企业页。招聘状态会变化，投递前请以打开后的企业页或岗位页实时信息为准。
