# 武汉跨境岗位发现工具

这是一个面向武汉跨境岗位的静态网页小工具。它不再依赖固定岗位详情页，而是用招聘 App 搜索入口、候选企业池和手动导入来持续发现新机会。

用途：

- 按你的简历能力、福利底线、公司规模、岗位类型和风险标签筛选武汉跨境岗位和公司。
- 公司数据拆到 `data/companies.json`、`data/candidate-jobs.json`、`data/rejected-companies.json`。
- 页面分为 `主投企业`、`待核验候选企业`、`已排除企业` 三块。
- 公司卡片显示当前状态、最后核验时间、推荐动作、公司规模、福利和风险标签。
- 公司卡片提供 BOSS、智联、猎聘搜索按钮，固定搜索“公司名 + 武汉 + 跨境电商运营”。
- 今日找新岗位入口固定搜索“武汉 跨境电商运营 双休 五险一金”。
- 按钮使用当前页面跳转，手机浏览器中更容易唤起招聘 App；没有唤起时进入对应平台搜索页。
- 支持“手动导入候选企业”：粘贴岗位文本后，自动分到待核验候选或已排除企业，并保存在当前浏览器本地。
- 公司卡片只保留 App 搜索和标记主投/待核验/排除，不再展示固定岗位详情链接。
- 记录每家公司投递状态和备注，数据保存在浏览器本地。
- 粘贴招聘 JD，输出“建议投 / 先问 / 不建议投”、风险词和可问 HR 的问题。
- 复制导出当前名单，方便发给自己或继续整理。

打开方式：

线上直接访问 GitHub Pages。若在本地预览，因为页面需要读取 `data/*.json`，请用静态服务器打开，例如：

```bash
python -m http.server 4173
```

然后访问 `http://127.0.0.1:4173/`。

数据说明：

公司池来自 2026-05-27 公开招聘页、校园就业网、企业介绍页、武汉跨境电商产业公开信息和招聘平台企业页。招聘状态会变化，投递前请优先用 BOSS、智联、猎聘 App 搜索入口核验最新在招岗位。批量导入的新候选只保存在本机浏览器，需要长期同步时可导出名单后整理进 `data/`。
