# 武汉跨境运营求职雷达

这是一个面向武汉跨境电商运营求职的本地/静态网页小工具。

用途：

- 按你的简历能力筛选武汉跨境电商运营公司。
- 公司名和“企业链接/企业搜索”可直接打开企业主页或预置搜索结果。
- 用 BOSS、智联、猎聘岗位入口替代容易过期的单个招聘详情页。
- 记录每家公司投递状态和备注，数据保存在浏览器本地。
- 粘贴招聘 JD，快速判断是否适合投。
- 复制导出当前名单，方便发给自己或继续整理。

打开方式：

直接用浏览器打开 `index.html`。不需要安装依赖。

数据说明：

公司池来自 2026-05-26 的公开招聘页、校园就业网、企业介绍页、武汉跨境电商产业公开信息和招聘平台搜索线索。招聘状态会变化，投递前请以 BOSS、智联、猎聘、公司官网或公众号实时信息为准。
